Digit Classification with TensorFlow

In this project I used TensorFlow to train a model that classifies images of digits into number classes.

About this project

This project required me to use Keras atop TensorFlow to train a model to classify a set of images into numbe classes.
I used the MNIST dataset for this single-label, multiclass classification problem. The dataset contained 70 000 samples, and I used 60 000 for training and 10 000 for testing.
The input to this model was images, and the output was a set of probabilities determining which number the images most likely contains.
